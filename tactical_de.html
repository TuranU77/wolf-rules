Taktik Modus (9–14 Spieler)

Was ihr braucht

  • 9–14 Spieler
  • 1 Moderator
  • Diese App verteilt die Rollen

Rollen

  • 🐺 Wölfe: Nachts gemeinsam ein Opfer auswählen
  • 🧙 Zauberer: Erkennt per Nicken/Kopfschütteln nachts Rollen
  • 🧪 Hexe: Tränke 1× nutzbar – Heilung oder Gift nach stummer Moderation
  • 🏹 Jäger: Beim Tod sofort auf jemanden zeigen
  • 🛡️ Wächter: Schützt jede Nacht jemanden (nicht 2× dieselbe Person)
  • 👧 Magd: Darf 1× am Tag eine Rolle öffentlich machen
  • 👨‍🌾 Dörfler: Keine Spezialfähigkeit

Spielablauf

Nacht

  1. Wölfe erwachen → Opfer wählen
  2. Zauberer → Untersuchung
  3. Wächter → Schutz
  4. Hexe → Heilung oder Gift

Tag

  1. Moderator nennt Todesopfer
  2. Jäger → ggf. Schuss
  3. Magd → ggf. Rolle aufdecken
  4. Diskussion → Abstimmung

Ziel

  • 🏘️ Dorf gewinnt: Alle Wölfe eliminiert
  • 🐺 Wölfe gewinnen: Gleichstand mit Dörflern
