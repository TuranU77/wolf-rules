Erweiterter Modus (6–15 Spieler)

Was ihr braucht

  • 6–15 Spieler
  • 1 Moderator
  • Diese App verteilt die Rollen

Rollen

  • 🐺 Wölfe: Wählen nachts ein Opfer.
  • 🧙 Zauberer: Zeigt nachts auf eine Person, Moderator antwortet stumm (Nicken/Kopfschütteln).
  • 🧪 Hexe: Hat einen Heil- und einen Gifttrank (je 1×). Moderator zeigt das Opfer, Hexe entscheidet stumm.
  • 🏹 Jäger: Stirbt → darf sofort auf jemanden zeigen, den er mitnimmt. Moderator nennt den Namen.
  • 👨‍🌾 Dörfler: Keine Spezialfähigkeit.

Nacht

  1. Wölfe wählen ihr Opfer
  2. Zauberer untersucht
  3. Hexe kann Tränke einsetzen

Tag

  1. Alle erwachen
  2. Moderator nennt die Opfer
  3. Jäger schießt ggf.
  4. Diskussion → Abstimmung

Ziel

  • 🏘️ Dorf gewinnt: Alle Wölfe eliminiert
  • 🐺 Wölfe gewinnen: Genauso viele Wölfe wie übrige Spieler
